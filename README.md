# 💰 Expense Tracker

A modern and responsive **Expense Tracker** web application built using **HTML, CSS, and JavaScript**. It helps users manage their finances by tracking income and expenses, setting monthly budgets, and visualizing spending patterns through interactive charts. All data is stored locally using the browser's **Local Storage**, eliminating the need for a backend.

---

## 🚀 Features

* Add income and expense transactions
* Delete individual transactions
* Track current balance, total income, and total expenses
* Set and monitor a monthly budget
* Visualize expenses with an interactive Pie Chart (Chart.js)
* Search and filter transactions
* Export transaction history to CSV
* Automatic data persistence using Local Storage
* Responsive design for desktop and mobile devices

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage API

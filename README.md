# Expense Tracker

A simple and user-friendly web application to manage daily expenses and track spending.

## Features
- Add and delete expenses  
- View total spending instantly  
- Simple and clean user interface  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Future Improvements
- Add budget tracking  
- Use local storage to save data  
- Add charts for better visualization
